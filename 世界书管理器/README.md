# 世界书管理器

在 SillyTavern 中集中管理全部世界书：虚拟文件夹、标签、筛选排序、条目编辑与批量操作。数据保存在**脚本变量** `wb_manager`，与酒馆原生世界书列表相互独立（排序/置顶不改变酒馆下拉框顺序）。

**入口**：酒馆右上角 **魔法棒**（`#extensionsMenu`）→ **世界书管理器**（`fa-book-open` 图标）。无需打开聊天即可使用。

**未纳入本脚本**：`世界书swipe管理器.ts`（请单独启用或停用）。

面向用户的分块说明见 **[功能介绍.md](./功能介绍.md)**（仿「世界书转预设」介绍体例）。

## 功能一览

### 打开与布局

| 能力 | 说明 |
|------|------|
| 全屏模态面板 | 桌面居中宽面板；手机全屏 + 底栏 Tab |
| 标题计数 | `显示 N / 共 M 本`（筛选结果 / 酒馆世界书总数） |
| 关闭 / 刷新 | 桌面在标题栏；手机在底栏左侧 |
| 虚拟滚动 | 世界书/条目列表，按内容估算**动态行高** |
| 实时同步 | 监听 `WORLDINFO_UPDATED`、`WORLDINFO_ENTRIES_LOADED`、`SETTINGS_UPDATED`，一般无需手动刷新页面 |

**桌面**：左栏文件夹 | 中栏世界书 | 右栏条目 + 工具栏。  
**手机**：顶栏文件夹横滑 → 搜索/排序/筛选 → 世界书 Tab 或条目 Tab；勾选世界书后底栏批量条 + FAB 更多操作。

### 虚拟文件夹

- 默认文件夹「未分类」不可删。
- **新建文件夹**：名称 + 关联标签（OR / AND）+ 图钉置顶。
- **点击文件夹**：按文件夹规则筛选世界书（关联标签 + OR/AND）。
- **拖拽排序**、**图钉置顶**（文件夹与世界书各自独立）。
- **齿轮编辑**（桌面侧栏 / 手机再点已选自定义文件夹）：改名称、标签、OR/AND。
- 世界书可手动归入文件夹；文件夹关联规则可自动匹配带标签的书。

### 标签

- **标签定义**：名称 + 颜色；搜索栏 **标签图标** 打开「标签管理」sheet（新增 / 改色 / 删除），筛选 sheet 内也有入口。
- **批量打标签 / 移除**：勾选多本后底栏「加标签」或 FAB「批量标签…」。
- **筛选**：筛选 sheet 内点标签芯片（OR：满足任一）；可与搜索、绑定、全局/当前角色卡、文件夹叠加。

### 筛选与搜索

- 搜索：世界书名称、标签等（输入约 280ms 防抖后生效）。
- 绑定：**全部 / 已绑定 / 未绑定**（未绑定 = 未出现在任何角色卡 primary/additional）。
- **仅全局开启**、**仅当前角色卡**（有聊天时）。
- 搜索框旁：**排序**、**筛选**（角标为活跃筛选数）、**标签**（标签管理）。

### 世界书列表

- **单击**（桌面）/ **点按**（手机）：选中并加载条目（手机会切到「条目」Tab）。
- **多选**（桌面）：勾选框；左侧把手拖拽排序。
- **多选**（手机）：**长按约 0.5 秒**。
- **图钉置顶**：置顶项固定在非置顶项之前（仅管理器内顺序）。
- **拖拽排序**（桌面）：调整 `bookOrder`。
- **右下角时间戳**（半透明）：`导入 yyyy-mm-dd` 或 `导入时间未知`（悬停可看说明；时间为插件记录或文件 Last-Modified，非精确导入时刻）。

### 世界书排序（仅管理器内）

排序方式（工具栏下拉 / 手机排序 sheet）：

| 分组 | 方式 |
|------|------|
| 导入 | 首次导入 旧→新 / 新→旧 |
| 修改 | 最近修改 旧→新 / 新→旧（管理器内 `lastTouchedAt`） |
| 名称 | A→Z / Z→A |
| 绑定 | 已绑定在前 / 未绑定在前 |
| 标签 | 按标签归类（同标签相邻） |

- **范围**：「仅排序筛选结果」或「排序全部」（忽略当前筛选）；无筛选时两者效果相同。
- **首次导入逻辑**：有 `firstImportAt` 的书按时间戳排序；无记录的书在排序「首次导入」时仍用启用管理器当日补写。
- **导入时间规则**：**首次打开管理器**时，当时已有的世界书统一记为**装插件当日**；此后在酒馆新加的世界书记**当前时刻**；管理器内复制亦记当前时刻。卡片显示「导入 yyyy-mm-dd」。不再读取文件 Last-Modified。
- **撤销**：可撤销操作后，标题栏或底栏会出现 **撤销** 按钮，点击即可撤回（不弹出 toastr 通知）。
- **日间/夜间**：标题栏太阳/月亮按钮切换，偏好保存在浏览器 `localStorage`（`wb-manager-theme`），与酒馆主题无关。
- **排序记忆**：上次使用的排序方式与范围会写入 `wb_manager`，桌面快速排序默认沿用上次范围。
- **插入到…下方**：勾选多本后，选锚点世界书，将选中块移到其正下方。

### 世界书批量与维护

| 操作 | 说明 |
|------|------|
| 全选 / 反选 / 取消勾选 | 针对当前筛选可见列表 |
| 勾选未绑定 | 一键选中所有未绑定角色卡的书 |
| 删除选中 | 未绑定直接删；已绑定列出角色卡后确认 |
| 复制 | 复制世界书内容 + **文件夹、标签等元数据**（不含置顶；新书 `firstImportAt` 为当前时间） |
| 重命名 | **须恰好勾选 1 本**；迁移绑定并更新 `bookOrder` |
| 归入文件夹 | 批量移动到目标文件夹 |

### 条目列表

- 预览：标题、UID、启用、概率、位置/深度、关键字摘要、内容摘要。
- **桌面**：左侧勾选框；**手机**：点条目主体切换选中，仅点「编辑」进入编辑页。
- **拖拽排序**（把手）：调整条目顺序并写回世界书。
- **插入到…下方**：多选条目后选锚点 UID。
- **全选 / 反选 / 取消勾选**。
- **删除**、**复制/移动到** 其它世界书（下拉选目标书）。
- 删除条目 / 删除世界书（确认后）：点面板 **撤销** 可撤回（进行其他会覆盖撤销的操作前有效）；撤销删除世界书时会恢复**角色卡 / 全局 / 聊天绑定**（在快照范围内）。

### 条目编辑

- 字段：标题、启用、策略（蓝/绿/向量）、触发率、插入位置/深度/顺序、关键字、递归选项、正文。
- **桌面**：居中对话框，字段多列紧凑布局，底部固定取消/保存。
- **手机**：底部 sheet，表单可滚动，底部固定取消/保存。
- 保存后写回当前世界书并刷新列表。

## 数据与持久化

- **脚本变量** `wb_manager`（`getVariables` / `replaceVariables`，`script_id` 为本脚本）。
- 主要内容：`folders`、`folderMeta`、`tagDefs`、`books`（每本元数据）、`bookOrder`、`filter`（筛选记忆）。
- 世界书正文与条目仍在酒馆世界书 JSON；导入时间 additionally 写入 `extensions.wm_first_import_at`（补写/首次导入排序时）。

## 使用提示

1. 在酒馆助手脚本库启用本脚本；建议开启「实时监听」以便开发时热重载。
2. 大量世界书时优先用**文件夹 + 标签 + 搜索**缩小范围，再排序或批量操作。
3. 管理器排序**不会**改变酒馆原生世界书选择器顺序；若需与酒馆一致，以酒馆界面为准。
4. 重命名后若个别角色卡附加世界书未更新，请在原生界面检查（脚本已通过 `updateCharacterWith` 尽力迁移）。

---

## 已知限制

- 装插件前旧书的「导入时间」仅为**启用管理器当日**的占位，不能还原真实历史顺序。
- 卡片标签超过 4 个时折叠为 `+N`（悬停可看完整列表）。
